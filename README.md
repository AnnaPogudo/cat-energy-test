1. Cat Energy — адаптивный лендинг

Адаптивный лендинг-магазин функционального питания для котов Cat Energy.
Вёрстка выполнена по макету Figma без использования CSS/JS-фреймворков.

Макет Figma:
https://www.figma.com/design/gqM9B9ZUTgcmVXsQ2W7V8C/Кэт-энерджи---landing--Copy-?node-id=2403-436&t=pohAEMXph1VdwMIP-0

2. Возможности проекта

-Полностью адаптивная вёрстка (мобильный → планшет → десктоп)
-Семантический HTML + доступность (ARIA, скрытые заголовки)
-Интерактивный слайдер «До/После» на чистом JavaScript
-Поддержка Retina-дисплеев (WebP + PNG)
-SVG-спрайт иконок
-Бургер-меню для мобильных устройств
-Яндекс.Карта через официальное API

3. Стек технологий

-HTML5
-CSS3 (Flexbox, Grid, медиазапросы)
-Vanilla JavaScript
